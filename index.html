About

Portfolio

Right Door

Right Door is a Metro route calculator with real-time train arrivals for the departing station. Comments allow users to share tips on where to board in order to be dropped closest to their next escalator. This was my first individual project; it was originally built with Sinatra and later reconstructed as I was learning Rails. Visit the code or read about the project on my blog.

TwitterBurst

TwitterBurst is a head-to-head, real-time hashtag competition built on Rails. I acted as project lead for this group project. My contributions included implementing the backend architecture, managing group workflow, and building a Node.js app that serves TwitterBurst with a custom Twitter stream for each match. When visiting this app, you may need to also awaken the Node server that is hosted for free on Heroku! Visit the code (Rails & Node) or read about the project on my blog.
